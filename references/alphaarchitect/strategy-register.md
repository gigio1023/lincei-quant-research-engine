# Alpha Architect Strategy Register

Status: supporting research register.

This register maps the ingested Alpha Architect corpus to testable strategy hypotheses for Lincei. It is not trading advice and it is not promotion evidence. Every row still needs point-in-time data, a LEAN implementation, cost/slippage assumptions, Cloud validation, current paper trading/shadow trading evidence, and reconciliation before self-funded capital allocation.

Corpus index: [index.json](index.json)

## Priority Labels

| Label | Meaning                                                             |
| ----- | ------------------------------------------------------------------- |
| `P1`  | Directly useful for the first self-funded capital research backlog. |
| `P2`  | Useful after baseline, data, or universe work exists.               |
| `P3`  | Context/risk note; do not implement as a direct strategy now.       |
| `Out` | Outside current allowed scope or needs a future spec.               |
