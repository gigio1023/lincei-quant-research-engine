# Local Reference Projects

This directory is for local cloned reference repositories only.

The clone directories are intentionally ignored by git. Keep this README tracked, but do not commit the cloned projects, their `.git` directories, build outputs, data, or logs.

Use these projects as references, not templates to copy blindly. Before using any pattern, write down:

- what problem the reference solves;
- which file or module was inspected;
- why the pattern fits this project;
- what we intentionally choose not to copy.

Current reference register: `../reference-register.md`.

Current pattern notes: `../reference-patterns.md`.
