# Phase 1 Plan

## Deliverables

- Baseline equal-weight backtest scaffold.
- Pydantic strategy and event schemas.
- Metrics for CAGR, volatility, Sharpe, max drawdown, turnover, and trade count.
- News/event extraction contract.
- No-live-trading risk gate.
- Markdown report renderer.
- Unit tests for schema, timestamps, metrics, and live gate.

## Not Yet

- Real market data ingestion.
- QuantConnect/LEAN integration.
- LLM API calls.
- ML models.
- Paper trading.
- Broker integration.

## Definition of Done

- `pytest` passes.
- Demo report can be generated from synthetic data.
- Docs explain purpose, architecture, risks, and references.
- Agent skill exists and can guide future development.
