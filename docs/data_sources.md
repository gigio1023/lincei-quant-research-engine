# Data Sources

No paid or broker-connected source is configured in this scaffold.

## Initial Allowed Data

- Local CSV adjusted close files for educational backtests.
- Synthetic data for tests and demos.
- Manually captured news/event examples with explicit timestamps.

## Future Candidates

- QuantConnect/LEAN datasets for hosted research.
- OpenBB integrations for research data access.
- SEC EDGAR filings.
- Earnings transcripts and macro calendars.
- Licensed news feeds only when source terms permit storage and processing.

Every source must document freshness, survivorship-bias risk, point-in-time availability, and license
constraints.
