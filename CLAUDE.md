# CLAUDE.md

## Project Identity

This is an AI-agent-assisted quant research and news/event intelligence system. The user is a strong
AI/software engineer and a finance beginner.

## How To Work

- Start from existing docs and Pydantic contracts.
- Keep modules small and testable.
- Explain finance assumptions when they appear in code.
- Write durable reports instead of chat-only conclusions when the result matters.
- Review strategy work as a skeptical quant researcher and risk officer.

## Safety Boundary

Default scope is research and paper-trading preparation only. Live trading, broker order submission,
leverage, margin, options, short selling, and crypto derivatives are blocked unless handled as a
separate explicit project with new risk documentation.

## News/NLP Direction

News and NLP are first-class components, but not direct trade triggers. Use LLMs for event extraction,
ticker mapping, materiality scoring, bull/bear/neutral thesis writing, contradiction checks, source
quality assessment, and report drafting. Validate outputs before they influence portfolio decisions.
