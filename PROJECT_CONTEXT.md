# Project Context

## Mission

Build an AI-agent-assisted quant research, news/event analysis, backtesting, and risk-review system
for aggressive but risk-budgeted personal wealth-growth experimentation.

## Operating Definition

Aggressive means faster research iteration, broader hypothesis generation, and stronger automation of
analysis. It does not mean uncontrolled execution, direct LLM-to-order pipelines, leverage, or hidden
broker access.

## User Profile

The user is strong in Python, backend systems, observability, LLM tooling, evaluation, and agentic
coding workflows. The system should assume software/AI fluency but explain finance concepts where
domain-specific assumptions matter.

## First Principles

- Research aggressively.
- Backtest skeptically.
- Treat news/NLP as a first-class research component.
- Treat LLM output as untrusted until validated.
- Paper trade before any real-money consideration.
- Keep human approval in the loop for anything execution-related.
