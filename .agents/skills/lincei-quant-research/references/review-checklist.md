# Review Checklist

## Critical

- Live trading path exists without a separate gate.
- Future data can enter signal generation.
- News ingestion time is ignored.
- LLM output can affect portfolio decisions without validation.

## Major

- Benchmark missing or weak.
- Costs, slippage, turnover, or drawdown missing.
- Universe introduces survivorship bias.
- Parameter search is not logged.
- Report omits limitations.

## Output Format

1. Critical issues.
2. Major issues.
3. Minor issues.
4. Required fixes before backtesting.
5. Required fixes before paper trading.
6. Final judgment.
